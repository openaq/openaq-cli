### api-key

```sh
openaq settings set api-key [api-key]
```

```sh
openaq settings get api-key
```

### format

```sh
openaq settings set format [json|csv|none]
```

```sh
openaq settings get format
```

### pretty

```sh
openaq settings set pretty [true|false]
```

```sh
openaq settings get pretty
```
